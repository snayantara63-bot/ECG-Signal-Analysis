# ECG-Signal-Analysis
Biomedical Signal Processing project: ECG signals are analyzed using Python. Time domain plots show electrical heart activity, and FFT (frequency domain) analysis identifies signal frequency components. Data sampled at 1 kHz.
# ECG Signal Analysis

This project analyzes ECG signals using Python.

## Features
- Time domain ECG signal visualization
- FFT frequency analysis
- ECG plots using Python

## Tools Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook
